## Tipos de Máscara Predefinidos

| Tipo | Formato | Ejemplo |
|------|---------|---------|
| `phone` | (999) 999-9999 | (555) 123-4567 |
| `phone-mx` | +52 (99) 9999-9999 | +52 (55) 1234-5678 |
| `credit-card` | 9999 9999 9999 9999 | 4111 1111 1111 1111 |
| `date` | 99/99/9999 | 25/12/2024 |
| `time` | 99:99 | 14:30 |
| `ssn` | 999-99-9999 | 123-45-6789 |
| `zip` | 99999 | 12345 |

## Máscara Personalizada

```html
<ui-input-mask
  label="Placa"

```

### Caracteres de Máscara

| Carácter | Descripción |
|----------|-------------|
| `9` | Solo dígitos (0-9) |
| `a` | Solo letras (a-z, A-Z) |
| `*` | Alfanumérico (letras y números) |

## Integración con Formularios

### Template-driven Forms

```html
<ui-input-mask
  [(ngModel)]="phone"
  maskType="phone"
  label="Teléfono"
></ui-input-mask>
```

### Reactive Forms

```typescript
form = new FormGroup({
  phone: new FormControl('', [Validators.required])
});
```

```html
<ui-input-mask
  formControlName="phone"
  maskType="phone"
  label="Teléfono"
  [invalid]="form.get('phone')?.invalid && form.get('phone')?.touched"
></ui-input-mask>
```
