## Eventos

| Evento | Descripción |
|--------|-------------|
| `onComplete` | Emitido cuando todos los campos están llenos |
| `onFocus` | Emitido al enfocar |
| `onBlur` | Emitido al perder el foco |

```html
<ui-input-otp
  [length]="6"
  (onComplete)="verifyCode($event)"
></ui-input-otp>
```
