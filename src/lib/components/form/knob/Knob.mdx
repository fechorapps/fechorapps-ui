## Rango Personalizado

```html

  [(ngModel)]="temperature"
  [min]="16"
  [max]="30"
  [step]="0.5"
  valueTemplate="{value}°C"
  label="Temperatura"
></ui-knob>
```

## Controles

| Interacción | Descripción |
|-------------|-------------|
| Arrastrar | Ajusta el valor siguiendo el movimiento del mouse |
| Scroll | Incrementa/decrementa el valor |
| Flechas | Ajusta el valor por pasos |
| Home/End | Va al mínimo/máximo |
| Page Up/Down | Ajusta el valor por 10 pasos |
