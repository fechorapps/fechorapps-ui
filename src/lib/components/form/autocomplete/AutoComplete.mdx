## Uso Básico

```typescript
// Definir sugerencias
suggestions: AutoCompleteSuggestion[] = [];

// Método de búsqueda
search(event: AutoCompleteCompleteEvent) {
  this.suggestions = this.items.filter(item =>
    item.label.toLowerCase().includes(event.query.toLowerCase())
  );
}
```

```html
<ui-autocomplete
  [(ngModel)]="selectedItem"
  [suggestions]="suggestions"
  (completeMethod)="search($event)"
  placeholder="Buscar..."
></ui-autocomplete>
```

## Interfaz de Sugerencias

```typescript
interface AutoCompleteSuggestion {
  label: string;      // Texto a mostrar
  value: unknown;     // Valor asociado
  disabled?: boolean; // Si está deshabilitado
  [key: string]: unknown; // Campos adicionales
}
```

## Eventos

| Evento | Tipo | Descripción |
|--------|------|-------------|
| `completeMethod` | `AutoCompleteCompleteEvent` | Se emite cuando se debe buscar |
| `onSelect` | `AutoCompleteSelectEvent` | Se emite al seleccionar un item |
| `onClear` | `void` | Se emite al limpiar la selección |
| `onShow` | `void` | Se emite al abrir el panel |
| `onHide` | `void` | Se emite al cerrar el panel |

```typescript
interface AutoCompleteCompleteEvent {
  query: string;
  originalEvent: Event;
}

interface AutoCompleteSelectEvent {
  value: AutoCompleteSuggestion;
  originalEvent: Event;
}
```

## Ejemplo con API

```typescript
@Component({...})
export class MyComponent {
  suggestions: AutoCompleteSuggestion[] = [];
  loading = false;

  async search(event: AutoCompleteCompleteEvent) {
    this.loading = true;
    try {
      const response = await this.api.search(event.query);
      this.suggestions = response.map(item => ({
        label: item.name,
        value: item.id
      }));
    } finally {
      this.loading = false;
    }
  }
}
```

```html
<ui-autocomplete
  [loading]="loading"
  [suggestions]="suggestions"
  (completeMethod)="search($event)"
></ui-autocomplete>
```

## Navegación por Teclado

| Tecla | Acción |
|-------|--------|
| `↓` | Siguiente opción / Abrir panel |
| `↑` | Opción anterior |
| `Enter` | Seleccionar opción resaltada |
| `Escape` | Cerrar panel |
| `Tab` | Cerrar panel y mover foco |

## Accesibilidad

- `role="combobox"` en el input
- `role="listbox"` en el panel
- `aria-expanded` indica estado del panel
- `aria-selected` en la opción resaltada
- `aria-disabled` en opciones deshabilitadas
- Navegación completa por teclado
