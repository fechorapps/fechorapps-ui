## Integración con Formularios

### Template-driven Forms

```html
<ui-input-text
  [(ngModel)]="email"
  name="email"
  label="Email"
  [invalid]="!isValid"
  helpText="Ingresa un email válido"
></ui-input-text>
```

### Reactive Forms

```typescript
form = new FormGroup({
  email: new FormControl('', [Validators.required, Validators.email])
});
```

```html
<ui-input-text
  formControlName="email"
  label="Email"
  [invalid]="form.get('email')?.invalid && form.get('email')?.touched"
  [helpText]="form.get('email')?.errors?.['email'] ? 'Email inválido' : ''"
></ui-input-text>
```

## Accesibilidad

- Labels asociados correctamente con `for`/`id`
- Estados `disabled` y `invalid` reflejados en atributos ARIA
- `aria-describedby` para texto de ayuda
- Focus visible con anillo de enfoque
