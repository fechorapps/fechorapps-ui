## Importación

```typescript
import { UiIconFieldComponent } from '@appcordion/ui';
```

## Uso Básico

```typescript
import { Search } from 'lucide-angular';
readonly searchIcon = Search;
```

```html
<ui-icon-field [icon]="searchIcon">
  <input type="text" placeholder="Buscar..." />
</ui-icon-field>
```
