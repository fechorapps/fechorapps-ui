## Paletas Personalizadas

```typescript
customPresets = [
  { name: 'Primary', value: '#1e3a5f' },
  { name: 'Secondary', value: '#2d4a6f' },
  { name: 'Accent', value: '#00b4d8' },
];
```

```html
<ui-color-picker
  [presets]="customPresets"
  label="Color del Tema"
></ui-color-picker>
```

## Opciones de Visualización

| Propiedad | Descripción |
|-----------|-------------|
| `showInput` | Muestra campo de entrada de texto |
| `showPresets` | Muestra la paleta de colores predefinidos |

```html
<ui-color-picker
  [showInput]="false"
  [showPresets]="true"
></ui-color-picker>
```
