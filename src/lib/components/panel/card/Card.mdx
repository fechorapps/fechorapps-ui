

## Proyección de Contenido

La tarjeta usa slots para contenido flexible:

```html
<ui-card title="Mi Tarjeta">
  <!-- Slot de header personalizado (sobreescribe title/subtitle) -->
  <ng-container header>
    <div class="flex items-center gap-2">
      <span class="text-2xl">🎯</span>
      <span>Header Personalizado</span>
    </div>
  </ng-container>

  <!-- Contenido principal (slot default) -->
  <p>Contenido principal aquí.</p>

  <!-- Slot de footer -->
  <ng-container footer>
    <ui-button>Acción</ui-button>
  </ng-container>
</ui-card>
```

## Accesibilidad

- Tarjetas clickables deben tener `role="button"` (agregado automáticamente)
- El contenido debe tener estructura semántica adecuada
- Imágenes deben incluir `imageAlt` descriptivo
- Focus visible en tarjetas interactivas

```html
<!-- Ejemplo accesible -->
<ui-card
  [clickable]="true"
  [hoverable]="true"
  imageUrl="course.jpg"
  imageAlt="Portada del curso de matemáticas"
  title="Matemáticas Avanzadas"
  subtitle="12 lecciones"
>
  <p>Descripción del curso...</p>
</ui-card>
```
